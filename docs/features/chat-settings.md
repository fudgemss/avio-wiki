# Chat Settings

Configure how Avio behaves in your server or chat.

## Notification Preferences

Choose where flight and booking alerts are sent:

- Current channel
- A specific channel
- Direct message

## Permissions

| Setting | Description |
|---------|-------------|
| `admin-only booking` | Restrict `/book` commands to admins |
| `public tracking` | Allow anyone to use `/flight` commands |
| `quiet hours` | Suppress non-urgent notifications during set hours |

!!! tip
    Use `/settings quiet-hours 22:00 07:00` to silence non-critical alerts overnight.
